Properties

declareColorsWithAllZeroStones

declareColorsWithAllZeroStones: boolean

Maintain the color key for colors which have zero stone for cells even for cells that have no stones at all. This indeed produces a 'cell' line for each cell of the board, which is not desirable for large board. Defaults to 'false'

declareColorsWithZeroStones

declareColorsWithZeroStones: boolean

Maintain the color key for colors which have zero stone for cells which have at least one stone. Defaults to 'false'

language

language: string

The error message output language

separators

separators: object

Different separator options

Type declaration

  • betweenColors: WhiteOption

    The separator to use between different color names in the same line. Defaults to 'space'

  • betweenCoordinates: WhiteOption

    The separator to use between different elements of a coordinate. Defaults to 'space'

  • betweenKeywords: WhiteWithNewlineOption

    The separator to use between each language keyword. Defaults to 'newline'

  • colorKeyToNumber: WhiteOption

    The separator to use between a color name and the number that follows. Defaults to 'tab'

  • keywordToCoordinates: WhiteOption

    The separator to use between the keyword and the first element of a coordinate. Defaults to 'space'

useFullColorNames

useFullColorNames: boolean

Use the full color name in output. Defaults to 'true'
