  • stringify: function
      • (board: Board, options?: Partial<GBBStringifyingOptions>): string

      • Convert the given JS Object representing a Board into a String representing a GBB board.

        throws

        GBBUnparsingErrors.InvalidSizeDefinition - If the board has cero or negative size in any width or height. GBBUnparsingErrors.HeadBoundaryExceeded - If the head position is negative or exceeds the board size in any of it's coordinates. GBBUnparsingErrors.InvalidBoardDefinition - If the board array has incorrect size (not matching the board defined size) in the main array or any of it's elements. GBBUnparsingErrors.InvalidCellDefinition - If any cell of the board array includes more than 4 keys or is missing any of 'a', 'n', 'r' or 'v' keys.

        Parameters

        • board: Board

          The Board to convert to a GBB board string.

        • Optional options: Partial<GBBStringifyingOptions>

          The options on how to produce the final string

        Returns string

        A String with a GBB Board format representation of the given board.
